# 🎯 Gênio X v4 — Sistema de Apoio à Aposta Consciente

O **Gênio X v4** é uma plataforma determinística avançada desenvolvida para auxiliar apostadores na gestão de banca, geração de cartelas e conferência de resultados. O sistema foca na **estratégia e controle financeiro**, transformando a aposta em um processo planejado e consciente.

---

## 🚀 Funcionalidades Principais

O sistema é dividido em dois módulos integrados que trabalham em conjunto para uma experiência completa:

### 1. Módulo Gênio X (Core)
*   **💰 Gestão de Capital:** Simulação de cenários reais com cálculo de **ISA (Índice de Stress de Aposta)** para evitar quebras de banca.
*   **🎰 Geração Determinística:** 3 modelos exclusivos (A, B, C) com filtros de sequências e pontuação de "score" para as melhores combinações.
*   **🔎 Conferência Automática:** Importe os resultados e confira instantaneamente o desempenho de todas as suas cartelas.

### 2. Módulo de Investimentos PRO
*   **📊 Dashboard de Controle:** Visão clara do saldo, total investido e prêmios recebidos.
*   **📈 Recuperação Progressiva:** Estratégia Khareis 226 para recuperação de perdas acumuladas.
*   **💾 Backup e Importação:** Salve seus dados em JSON para nunca perder seu histórico de apostas.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido com foco em portabilidade e performance, utilizando tecnologias web puras:
*   **HTML5:** Estrutura semântica e interface intuitiva.
*   **CSS3:** Design responsivo e moderno (Mobile-First).
*   **JavaScript (Vanilla JS):** Lógica de processamento local, sem dependências externas ou bancos de dados pesados.

---

## 📖 Como Utilizar

Para rodar o sistema, você não precisa instalar nada. Basta clonar este repositório e abrir o arquivo principal no seu navegador:

1.  Abra o arquivo `index.html` (Gênio X).
2.  Utilize o botão **"🏦 Ir para Investimentos"** para acessar o módulo de gestão financeira.
3.  Utilize o botão **"🎯 Voltar ao Gênio X"** para retornar à geração de cartelas.

> **Dica:** Se estiver usando o GitHub Pages, o sistema funcionará como um Web App diretamente pelo link do repositório.
